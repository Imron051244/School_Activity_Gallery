# 🏫 School Activity Gallery (ระบบคลังภาพกิจกรรมโรงเรียน)

ระบบคลังภาพกิจกรรมและดาวน์โหลดภาพสำหรับสถานศึกษา พัฒนาด้วย React และ TailwindCSS ทำงานร่วมกับ Google Sheets และ Google Drive ในรูปแบบ Serverless (ผ่าน Google Apps Script) ออกแบบมาเพื่อให้ผู้ใช้งานทั่วไปและแอดมินใช้งานได้อย่างรวดเร็ว มีประสิทธิภาพสูง และประหยัดแบนด์วิดท์อินเทอร์เน็ต

---

## ✨ คุณสมบัติเด่นของระบบ (Features)

*   **⚡ High-Speed Image Loading (โหลดรูปภาพเร็วขึ้น 10 เท่า):**
    *   แปลงลิงก์ Google Drive อัตโนมัติเป็น Google Drive Thumbnail API ขนาด `w400` สำหรับหน้าตาราง และ `w1600` สำหรับหน้าขยายรูปเต็มจอ (Lightbox) ลดขนาดไฟล์ดาวน์โหลดจาก 5-10MB เหลือเพียง 15-30KB ต่อรูป
    *   ใช้คุณสมบัติ `decoding="async"` ป้องกันเบราว์เซอร์หน่วงค้างขณะเลื่อนชมแกลเลอรี
*   **🛠️ Persistent & Flexible API Configuration (การจัดการ API อัจฉริยะ):**
    *   **โหมดจำลองเริ่มต้น (Offline/Mock Mode):** สำหรับผู้ใช้ใหม่ที่เปิดเว็บครั้งแรก ระบบจะแสดงภาพและหมวดหมู่ตัวอย่างทันทีโดยไม่ติดขัดหรือมีหน้าต่างป๊อปอัปแจ้งเตือนกวนใจ
    *   **บันทึกถาวรเฉพาะเครื่องแอดมิน:** เมื่อแอดมินนำ URL ของ Google Apps Script มากรอกและบันทึกในหน้าตั้งค่า ระบบจะจดจำข้อมูลนั้นผ่าน LocalStorage เฉพาะในเครื่องนั้นตลอดไป ทำให้ไม่ต้องคัดลอกมาวางใหม่ทุกครั้ง
*   **📂 Folder-based Sync (ระบบซิงค์ไฟล์อัจฉริยะ):**
    *   รองรับการแบ่งหมวดหมู่กิจกรรมย่อยและโฟลเดอร์ย่อยใน Google Drive
    *   แอดมินสามารถกดปุ่ม **"ซิงค์รูปภาพจาก Google Drive"** เพื่อตรวจสอบ เปรียบเทียบ และเพิ่ม/ลบรายชื่อรูปภาพจากโฟลเดอร์ Drive เข้าสู่ระบบฐานข้อมูลชีตได้ในคลิกเดียว
*   **📥 Bulk & Individual Download (ระบบดาวน์โหลดภาพ):**
    *   ดาวน์โหลดรูปภาพเดี่ยว หรือเลือกรูปภาพหลาย ๆ รูปแล้วดาวน์โหลดพร้อมกันเป็นชุด (Bulk Download) ได้อย่างสะดวกสบาย
*   **🎨 Dynamic School Branding (ปรับเปลี่ยนแบรนด์ได้อิสระ):**
    *   เปลี่ยนชื่อสถานศึกษา คำบรรยายใต้ภาพ และอัปโหลดโลโก้โรงเรียนใหม่ได้โดยตรงจากหน้าควบคุมการตั้งค่า

---

## 💻 เทคโนโลยีที่ใช้งาน (Tech Stack)

*   **Frontend:** React, TailwindCSS, Axios, TanStack Query (React Query), Framer Motion (สำหรับแอนิเมชัน), SweetAlert2, Webpack
*   **Database & API Backend:** Google Sheets + Google Drive ทำงานร่วมกับ Google Apps Script (REST API JSON Endpoint)

## ⚙️ วิธีการติดตั้งและใช้งาน (Installation & Setup)

### 1. การเตรียมระบบหลังบ้าน (Backend - Google Sheets & Apps Script)
1.  สร้าง Google Spreadsheet ใหม่ขึ้นมา 1 ไฟล์
2.  ไปที่เมนู **ส่วนขยาย (Extensions)** > **Apps Script**
3.  ลบโค้ดเริ่มต้นในไฟล์ `Code.gs` ออกให้หมด
4.  คัดลอกโค้ดทั้งหมดจากไฟล์ [google-apps-script.gs](./google-apps-script.gs) ในโปรเจกต์นี้ไปวางแทนที่
5.  กด **บันทึก (Save)**
6.  คลิกปุ่ม **การทำให้ใช้งานได้ (Deploy)** > **การทำให้ใช้งานได้ใหม่ (New deployment)**
7.  เลือกประเภทการทำงานเป็น **เว็บแอป (Web app)**
8.  ตั้งค่าตัวเลือกดังนี้:
    *   *Execute as:* `Me` (ตัวฉันเอง)
    *   *Who has access:* `Anyone` (ทุกคน) **(สำคัญมาก: เพื่อให้ระบบเว็บแอป React สามารถเข้าถึงข้อมูลได้)**
9.  กด **Deploy** จากนั้นคัดลอก **Web App URL** ที่ได้ไว้เพื่อนำไปใช้ในขั้นตอนถัดไป

### 2. การติดตั้งฝั่งหน้าบ้าน (Frontend - React)
ติดตั้งเครื่องมือและรันระบบเซิร์ฟเวอร์บนเครื่องเครื่องของคุณ:

```bash
# 1. ติดตั้ง Dependencies ทั้งหมดในโปรเจกต์
npm install

# 2. เริ่มรันระบบสำหรับพัฒนา (Development Server)
npm run dev
```

เปิดบราวเซอร์เข้าสู่ลิงก์: [http://localhost:8080](http://localhost:8080)

## 🔧 วิธีการเชื่อมโยงระบบเข้ากับ Google Sheets
1.  เปิดหน้าแอปพลิเคชันหลัก เลือกเมนู **"ตั้งค่า" (Settings)** จากแถบเมนูด้านซ้าย
2.  ในส่วน **"กำหนดค่า API (เชื่อมต่อจริง)"** ให้กดเปิดสวิตช์ **"สถานะการเชื่อมต่อจริง"**
3.  นำลิงก์ **Web App URL** ของ Google Apps Script ที่ได้จากขั้นตอนก่อนหน้ามาวางลงในช่องป้อนข้อมูล
4.  กดปุ่ม **"บันทึกการตั้งค่า API"**
5.  ระบบจะเชื่อมต่อไปยังฐานข้อมูลจริงของสถานศึกษาให้ทันที!
